# My 90 Days of DevOps Learning Plan

## Current Level

I am a fresher Holds Bachelors of computer science degree with a strong interest in DevOps, Cloud Computing, and AI. I have basic knowledge of Linux, Git, and GitHub and am currently building my foundation through hands-on learning and projects.

## My Understanding of DevOps & Cloud Engineering

- DevOps is continuous deployment & testing of code. It is integration of development and operation to deliver faster software, more securely and more efficiently/ 

- Cloud Engineering involves designing, deploying, and managing applications and infrastructure on cloud platforms such as AWS. It enables organizations to build scalable, reliable, and cost-effective solutions.

## Why I Am Learning DevOps & Cloud

- I am Learning DevOps and Cloud because As a fresher I want to Perfect One Skill and for that DevOps & Cloud is best option because I enjoy solving technical problems, automating repetitive tasks, and working with modern technologies that power real-world applications. My goal is to build industry-relevant skills and prepare myself for a career as a DevOps Engineer.

## Where I Want to Reach

By the end of these 90 days, I want to:

1. Build a strong foundation in Linux, Git, Networking, and Cloud fundamentals.
2. Create and deploy real-world DevOps projects using Docker, Kubernetes, and AWS.
3. Become confident enough to apply for DevOps and Cloud-related fresher roles and internships.

## 3 Core DevOps Skills I Want to Build

1. Linux administration and troubleshooting.
2. CI/CD pipeline creation and automation.
3. Containerization and orchestration using Docker and Kubernetes.

## How I Will Stay Consistent 

- Follow the roadmap daily without skipping learning sessions.
- Maintain notes and document my progress on GitHub.
- Share key learnings and updates on LinkedIn.

- Track my progress weekly and adjust my learning plan when necessary.

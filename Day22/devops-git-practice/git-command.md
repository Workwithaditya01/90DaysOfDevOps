# Day 22 – Introduction to Git: Your First Repository

## Setup & Config

### Git Version:

```command
git --version
```

### Setting up Git identity:

```command
git config --global user.name "Workwithaditya01"
git config --global user.email "workwithaditya01@gmail.com"
```

### Git config:

```command
git config --list
```
---

## Git Workflow

### Initializing git:

```command
git init
```

### Staging file for commit

```command
git add
```

### Saving Staged changes.

```command
git commit -m "Commit msg"
```
---

## View Changes

### Display repository status

```command
git status
```

### Display commit history:

```command
git log
```

### Shows compact commit history:

```command
git log --oneline
```

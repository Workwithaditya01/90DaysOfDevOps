# 🚀 90 Days of DevOps Challenge

Welcome to my **90 Days of DevOps Challenge**! 🎯

This repository documents my journey of learning **DevOps from the ground up** through daily hands-on practice, projects, and notes. The goal is to build a strong foundation in DevOps tools, cloud technologies, automation, CI/CD, and infrastructure while maintaining consistency for 90 consecutive days.

---

## 🎯 Challenge Goals

- Build a strong DevOps foundation
- Learn industry-standard DevOps tools
- Practice hands-on every day
- Create real-world projects
- Document everything publicly
- Become job-ready by the end of the challenge
